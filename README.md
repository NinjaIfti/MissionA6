# SwiftCart

## প্রশ্নোত্তর (Bangla)

**১) null এবং undefined এর পার্থক্য কি?**

null মানে ইচ্ছাকৃতভাবে কোনো মান নেই বা খালি। undefined মানে ভেরিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু মান দেওয়া হয়নি অথবা কোনো প্রপার্টি/ভেরিয়েবল অস্তিত্বে নেই।

**২) JavaScript এ map() এর কাজ কি? forEach() থেকে কিভাবে আলাদা?**

map() অ্যারেতে প্রতিটি উপাদানের উপর ফাংশন চালিয়ে একটা নতুন অ্যারে রিটার্ন করে। forEach() শুধু লুপ চালায়, কিছু রিটার্ন করে না।

**৩) == এবং === এর পার্থক্য কি?**

== টাইপ কনভার্শন করে তারপর তুলনা করে। === টাইপ ও ভ্যালু দুটোই একই হতে হবে, কোনো কনভার্শন নেই।

**৪) API ডেটা ফেচ করতে async/await এর গুরুত্ব কি?**

async/await দিয়ে অ্যাসিঙ্ক্রোনাস কোড সিঙ্ক্রোনাসের মত লিখা যায়। API কলের পর রেসপন্স আসা পর্যন্ত অপেক্ষা করে তারপর পরের লাইন রান হয়, তাই ডেটা নিয়ে কাজ করা সহজ হয়।

**৫) JavaScript এ Scope বলতে কি বোঝায়? (Global, Function, Block)**

Global scope মানে ফাইল বা স্ক্রিপ্ট জুড়ে ভেরিয়েবল একসেস করা যায়। Function scope মানে ভেরিয়েবল শুধু সেই ফাংশনের ভেতরে থাকলে একসেস করা যায়। Block scope মানে curly braces {} এর ভেতরে ডিক্লেয়ার করা let/const শুধু সেই ব্লকের ভেতরেই থাকে।
